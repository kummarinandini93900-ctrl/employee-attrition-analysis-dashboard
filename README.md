# Employee Attrition Analysis Dashboard

## Project Overview
This dashboard analyzes employee attrition trends across departments using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## KPI Metrics
- Total Employees: 100
- Attrition Count: 31
- Attrition Rate: 31%
- Average Salary: ₹58,891

## Key Insights
- IT department has the highest attrition count.
- Average salary varies across departments.
- Interactive slicers allow filtering by gender and department.
